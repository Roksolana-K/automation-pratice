### 📚 Homework 1  

**Task:**  
- Open the WebdriverIO main page;
- navigate to the API page;
- verify the URL;
- verify H1 says “Introduction”;
- check breadcrumbs to contain "Introduction";
- check WebDriver link href;
- use search input, enter and clear text.

**Task:**
